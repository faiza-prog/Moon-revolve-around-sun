**🌙 Moon Revolve Around Sun**
A beautiful animation showing the moon revolving around the sun using HTML and  CSS
**✨ Features**
🎨 Smooth CSS animations
🌙 Realistic moon orbit simulation
📱 Responsive design
⚡ Lightweight and fast
**🛠️ Technologies Used**
HTML5 - Structure
CSS3 - Styling and animations
**💡 How It Works**
The animation uses:
CSS @keyframes for smooth circular motion
Transform properties for rotation
Absolute positioning for orbital path
